Star Wars Survey Classification Analysis

A Machine Learning Classification Project


Introduction

Given a Star Wars Survey dataset, our aim is to train a classifier that can predict a person's responses to questions about the Star Wars franchise given their answers to other such questions, or to try to predict the person's personal characteristics based off of their responses to questions about the franchise.

Dataset Description

The Star Wars Survey dataset is a labeled dataset with 1188 records. It has 15 features, which are survey questions regarding people's opinions on the Star Wars franchise and some personal information such as:

  • "Do you consider yourself a Star Wars fan?",
  • "Which is your favorite movie?",
  • "Which character shot first?",
  • "Gender",
  • "Income",
  • et cetera.

More information about this dataset can be found here.

Miscellaneous Files

  • Column Names - Text file manually created and used in the Data Preparation notebook to replace the two header rows of the dataset with a simpler, single header row.
  • Unique Values - Text file generated in the Data Preparation notebook in order to manually check for inconsistent values in the dataset.
